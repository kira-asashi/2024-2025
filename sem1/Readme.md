# About the program
### Grade Computation
* Midterm Grade
* Final Grade

The program allows users to input the amount of quiz/activity and their grade and it will calculate your midterm grade and final grade using FEU TECH's grading system:</br>

Without Lab:</br>

Midterm Grade (MG) = 60% CS + 40% ME</br>
Final Grade (FG) = 60% CS + 15% ME + 25% FE</br>

With Lab:</br>

Midterm Grade (MG) = 70% (Lecture Grade | 60% CS + 40% ME) + 30% (Lab Grade | 60% CS + 40% ME)</br>
Final Grade (FG) = 70% (Lecture Grade   | 60% CS + 15% ME + 25% FE) + 30% (Lab Grade | 60% CS + 15% ME + 25% FE)</br></br>

The program calculates the average of Short Quizzes as a single/additional Long Quiz
